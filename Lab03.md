# 🤖 DSP Lab 3: The Machine That Listens (Speech Recognition)

ยินดีต้อนรับสู่ภารกิจที่ 3 ของปฏิบัติการ Digital Signal Processing!
จากสัปดาห์ที่แล้วที่เราได้มองเห็นเสียงในรูปแบบของภาพ (Spectrogram) สัปดาห์นี้เราจะก้าวไปอีกขั้น ด้วยการนำภาพเหล่านั้นไปสอนให้คอมพิวเตอร์ "ฟัง" และ "เข้าใจ" ภาษามนุษย์ ผ่านการสร้างโครงข่ายประสาทเทียม (Artificial Neural Network) ของตัวคุณเอง!

## 🎯 วัตถุประสงค์การเรียนรู้ (Mission Objectives)
1. เข้าใจความสำคัญของ Mel-Scale ที่มีต่อการปรับแต่งความถี่เสียงให้คล้ายคลึงกับการรับรู้ของหูมนุษย์
2. สามารถสกัดคุณลักษณะเด่น (Feature Extraction) จากคลื่นเสียงให้กลายเป็นภาพ Mel-Spectrogram ได้
3. **[Highlight]** เข้าใจสถาปัตยกรรมและสามารถสร้างโมเดลสมองกล Convolutional Neural Network (CNN) เบื้องต้นสำหรับการจำแนกเสียงพูดได้
4. เข้าใจหลักการของ Data Consistency (การจัดเตรียมขนาดข้อมูลให้เท่ากัน) ก่อนนำไปป้อนให้ AI
5. **[Advanced]** สามารถนำโมเดลที่ฝึกสอนเสร็จแล้ว (Trained Model) มาทดสอบกับเสียงพูดจริงของตนเองแบบ Real-time บนระบบได้
