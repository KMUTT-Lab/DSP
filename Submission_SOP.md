# 🚀 SOP: ขั้นตอนการส่งงาน DSP Lab (Submission Guide)

คู่มือฉบับนี้คือแนวทางปฏิบัติมาตรฐาน (Standard Operating Procedure) สำหรับการส่งงานปฏิบัติการ DSP ในทุกสัปดาห์ ขอให้ทำตามขั้นตอนอย่างเคร่งครัดเพื่อให้ผลงานของคุณถูกบันทึกเข้าสู่ระบบอย่างสมบูรณ์

---

## 🛠️ Phase 1: การบันทึกงานลง GitHub (Save & Commit)
เมื่อคุณทำ Lab บน Google Colab เสร็จสิ้นและตรวจสอบผลลัพธ์ (กราฟ/เสียง) เรียบร้อยแล้ว ให้ดำเนินการดังนี้:

1. ไปที่เมนูแถบด้านบนใน Google Colab คลิกที่ **`File`** (ไฟล์)
2. เลือก **`Save a copy in GitHub`** (บันทึกสำเนาใน GitHub)
3. *(หากเป็นการใช้งานครั้งแรก)* ระบบจะหน้าต่างขึ้นมาขอสิทธิ์ ให้กด **Authorize** เพื่ออนุญาตให้ Colab เข้าถึงบัญชี GitHub ของคุณ
4. ในหน้าต่างตั้งค่าการบันทึก ให้ระบุข้อมูลดังนี้:
   * **Repository:** เลือก Repository ที่คุณใช้เก็บงานวิชานี้ (เช่น `ชื่อผู้ใช้/DSP-Labs`)
   * **Branch:** `main`
   * **File path:** ตั้งชื่อไฟล์ตามรูปแบบที่กำหนด เช่น `Lab01_67XXXXXXX.ipynb` (เปลี่ยนเป็นรหัสนักศึกษาของตนเอง)
   * **Commit message:** พิมพ์ข้อความสั้นๆ เช่น `Completed Lab 1` หรือ `Finished Audio Magic`
5. กดปุ่ม **`OK`** เพื่อทำการ Push ไฟล์ขึ้นระบบ
6. ระบบจะเปิดหน้าต่าง GitHub ใหม่ขึ้นมาโดยอัตโนมัติ เพื่อแสดงไฟล์ที่คุณเพิ่งบันทึก

---

## 🔗 Phase 2: การคัดลอกลิงก์ผลงาน (Get the URL)
เพื่อให้สามารถเข้าไปประเมินผลงานได้ คุณต้องนำลิงก์ที่ถูกต้องมาส่ง:

1. ตรวจสอบให้แน่ใจว่า Repository ของคุณตั้งค่าเป็น **Public** (หากเป็น Private ระบบจะไม่สามารถเข้าถึงไฟล์ได้)
2. ในหน้าต่าง GitHub ที่เปิดไฟล์ `.ipynb` ของคุณอยู่ ให้คลิกที่ **Address bar** ด้านบนสุดของเบราว์เซอร์
3. **คัดลอก (Copy) URL ทั้งหมด** *(ตัวอย่างลิงก์ที่ถูกต้อง: `https://github.com/Username/DSP-Labs/blob/main/Lab01_67XXXXXXX.ipynb`)*

---

## 📥 Phase 3: การส่งงานเข้าสู่ระบบ (Form Submission)
ขั้นตอนสุดท้ายคือการนำลิงก์มายืนยันในระบบฐานข้อมูล:

1. เข้าสู่ **[ลิงก์ Google Form สำหรับส่งงาน DSP Lab]** *(ลิงก์ฟอร์มส่งให้ใน class ครับ)*
2. กรอกข้อมูลส่วนตัวให้ครบถ้วน:
   * รหัสนักศึกษา (Student ID)
   * ชื่อ-นามสกุล (Full Name)
   * กลุ่มปฏิบัติการ (Section)
3. เลือก **หัวข้อ Lab** ที่ต้องการส่งให้ถูกต้อง
4. วาง URL ที่คัดลอกมาจาก Phase 2 ลงในช่อง **"Direct Link to .ipynb file"**
5. **ส่วนสำคัญ (Self-Reflection):** ตอบคำถามเกี่ยวกับการเรียนรู้ของคุณในสัปดาห์นั้น
   * อะไรคือความท้าทายที่สุดที่คุณพบ?
   * วันนี้คุณได้เรียนรู้อะไรใหม่บ้าง?
   * ประเมินความเข้าใจของตนเอง (1-5)
6. ตรวจสอบความถูกต้องทั้งหมด แล้วกด **`Submit`** (ส่ง)
